# FAQ: AI Providers

## Можно ли использовать ChatGPT вместо Claude?

**Да!** Coherent поддерживает несколько AI провайдеров:

1. **Anthropic Claude** (по умолчанию)
2. **OpenAI/ChatGPT** (опционально)
3. **Автоматическое обнаружение** (использует доступный провайдер)

### Как использовать ChatGPT:

1. **Установите пакет OpenAI:**
   ```bash
   npm install openai
   ```

2. **Настройте API ключ:**
   ```bash
   export OPENAI_API_KEY=your_openai_key
   # или в .env файле
   echo "OPENAI_API_KEY=your_openai_key" > .env
   ```

3. **Coherent автоматически обнаружит и использует OpenAI**, если ключ доступен.

## Можно ли автоматически подключиться к AI из Cursor?

**Частично.** Coherent пытается автоматически обнаружить API ключи из окружения IDE:

### Поддерживаемые переменные окружения:

- `OPENAI_API_KEY` - общий OpenAI ключ
- `CURSOR_OPENAI_API_KEY` - ключ из Cursor IDE
- `GITHUB_COPILOT_OPENAI_API_KEY` - ключ из GitHub Copilot
- `ANTHROPIC_API_KEY` - ключ Claude

### Как это работает:

1. Coherent проверяет переменные окружения в порядке приоритета
2. Если найден `OPENAI_API_KEY` или `CURSOR_OPENAI_API_KEY` → использует OpenAI
3. Если найден `ANTHROPIC_API_KEY` → использует Claude
4. Если ничего не найдено → показывает ошибку с инструкциями

### Ограничения:

**Cursor может не экспортировать ключи в дочерние процессы.** В этом случае:

1. **Проверьте доступность ключа:**
   ```bash
   echo $CURSOR_OPENAI_API_KEY
   ```

2. **Если ключ недоступен:**
   - Используйте настройки Cursor для экспорта переменных окружения
   - Или установите `OPENAI_API_KEY` вручную в `.env` файле

## Какой провайдер лучше?

| Провайдер | Плюсы | Минусы |
|-----------|-------|--------|
| **Claude** | Отличное качество кода, сильный structured output | Требует отдельный API ключ |
| **OpenAI** | Широко доступен, может быть уже настроен в IDE | Может требовать установку пакета |

**Рекомендация:** Используйте тот, который у вас уже настроен. Если у вас есть ключ в Cursor - используйте OpenAI. Если нет - используйте Claude.

## Можно ли использовать оба провайдера?

**Да, но не одновременно.** Coherent использует один провайдер за раз:

- Если доступны оба ключа, приоритет у OpenAI (проверяется первым)
- Можно переключаться, меняя переменные окружения

## Безопасность

- ✅ API ключи **не передаются** третьим лицам
- ✅ Ключи используются **только для генерации кода**
- ✅ Каждый пользователь использует **свой ключ**
- ⚠️ **Не коммитьте** `.env` файлы в Git
