# Story 3.12: Import CLI Flow (Figma End-to-End)

## Status

Draft

---

## Story

**As a** user,  
**I want** a single flow: `coherent import figma <url> --token <token>` that fetches, extracts tokens, normalizes components, and generates pages, then reports clearly,  
**so that** I can run `coherent preview` and `coherent chat` immediately after import.

---

## Acceptance Criteria

- [ ] Full import flow works end-to-end (fetch → tokens → components → pages)
- [ ] Post-import, all coherent commands work (chat, preview, validate, export)
- [ ] Design System viewer shows imported tokens and components
- [ ] CLI output matches spec: Fetching… Found N pages, M components… Extracting tokens… Normalizing components… Generating pages… Import complete!

---

## Details (from Epic 3)

**Example output:**

```
$ coherent import figma https://www.figma.com/file/abc123/MyDesign --token figd_xxx

🔍 Fetching Figma file "MyDesign"...
📦 Found: 5 pages, 12 components, 8 color styles, 4 text styles

🎨 Extracting design tokens...
   ✅ 5 colors → CSS variables
   ✅ 4 text styles → typography scale
   ✅ Border radius: 8px → --radius: 0.5rem

🧩 Normalizing components...
   ✅ Button (3 variants) → base Button
   ✅ Card → base Card
   ✅ PricingTier → CID-001 (shared section)
   ...

📄 Generating pages...
   ✅ Home → app/page.tsx
   ✅ Pricing → app/pricing/page.tsx
   ...

✅ Import complete!
   Run: coherent preview
   Then: coherent chat "make the primary color darker"
```

Post-import: preview, chat, validate, export all work with existing workflow.
