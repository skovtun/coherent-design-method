# Story 3.1: coherent export

## Status

In Progress

---

## Story

**As a** user who has generated a prototype with Coherent,  
**I want** to run `coherent export` and get a clean, deployable Next.js project without platform overlay,  
**so that** I can ship the result to Vercel, Netlify, or any host.

---

## Acceptance Criteria

- [ ] Exported project builds without errors
- [ ] No Design System routes/pages in export (unless `--keep-ds`)
- [ ] No `coherent.components.json` in export
- [ ] No floating DS button in export
- [ ] Shared components work correctly in export
- [ ] `next build` passes in exported directory

---

## Details (from Epic 3)

**Command:** `coherent export --output ./my-app`

**Behavior:**

1. Copy all app code (pages, components, styles, public assets)
2. **Strip platform overlay:** remove floating Design System button; remove `/design-system/*` routes and pages; remove `/api/design-system/*` API routes; remove DS layout, nav, components
3. **Strip Coherent-specific:** `coherent.components.json`, `design-system.config.ts` (unless kept), any `{/* Platform overlay */}` marked components
4. **Keep:** all user pages, shared components, `components/ui/*`, `globals.css`, `tailwind.config.ts`, `package.json` (cleaned of coherent-specific deps)
5. Run `next build` in output directory to verify (unless `--no-build`)
6. Report: Exported to X, Pages: N, Components: N, Build: success, Deploy hints

**Flags:** `--output <dir>` (default `./export`), `--no-build`, `--keep-ds`
